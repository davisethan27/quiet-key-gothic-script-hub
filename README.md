## What It Does

The Quiet Key is a companion utility built around lockpicking support for Gothic remake sessions. Its workflow centers on a lightweight overlay that can present lock family recognition, tension waveform feedback, and context-sensitive guidance while you handle locks in game.

It is intended for players who want a more readable view of lock behavior and a better way to react to different lock profiles. The toolkit includes multilingual UI support, preset-based behavior, optional AI hint integration, and accessibility-minded options to make it easier to use in different setups.

## Script Features

- Lock family recognition for identifying related lock patterns
- Tension waveform visualization to help interpret lock behavior
- Multilingual interface for broader language support
- Profile-aware presets for different play styles or game contexts
- Contextual hints that respond to the current lock interaction
- Accessibility support for more flexible on-screen use
- Optional AI hint integration for additional guidance
- Non-invasive overlay designed to sit alongside gameplay
